wcc.videolink Installation
--------------------------

To install wcc.videolink using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``wcc.videolink`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        wcc.videolink

* Re-run buildout, e.g. with:

    $ ./bin/buildout
